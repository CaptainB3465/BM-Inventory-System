 BM Inventory System

BM Inventory System is a simple Java-based inventory management application that helps users track and manage products, stock levels, and transactions efficiently.

Features

- Add, update, and delete products  
- Track product quantities in stock  
- Simple command-line interface for easy use  
- Generate basic inventory reports  

 Technologies Used

- Java  
- Git for version control  
- Optional: Database (if implemented)
